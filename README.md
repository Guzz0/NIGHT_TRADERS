## Descripción del proyecto

Nuestro proyecto se enfocó en analizar y predecir el cierre de huecos nocturnos bursátiles utilizando diferentes índices bursátiles y variables financieras. Después de obtener los datos de Yahoo Finance, se evaluaron diversas combinaciones de índices y variables, implementando una función para mejorar las predicciones. Se llevaron a cabo pruebas con diversos modelos de aprendizaje automático. Se aplicó este modelo en un conjunto de datos final compuesto por los últimos 30 días de movimientos bursátiles y se creó un dashboard en Tableau para facilitar el acceso a las predicciones del modelo.
